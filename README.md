# Job Portal Backend System

A role-based job portal backend application built with **Spring Boot**, **MySQL**, **JWT**, and **Docker**. This project provides secure APIs for recruiters and candidates to manage job postings, applications, and profiles efficiently.

---

## Features

- **Role-Based Access:** Separate authentication and access for recruiters and candidates.
- **JWT Authentication:** Secure login and session management using JSON Web Tokens.
- **Relational Database:** MySQL with JPA for storing users, jobs, and applications.
- **Dockerized Deployment:** Run the backend consistently on any environment using Docker.
- **REST APIs:** Fully functional APIs for job posting, job searching, and user management.

---

## Tech Stack

- **Backend:** Spring Boot
- **Database:** MySQL
- **Authentication:** JWT
- **Deployment:** Docker
